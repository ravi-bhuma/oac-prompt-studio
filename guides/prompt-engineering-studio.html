    🏷️
    1. Attribute Grouping
    Category: Behavior
    💡 Why This Matters
    Your data columns often have too many specific values that don't match how people actually talk. This instruction tells the agent to automatically group raw values into business-friendly categories.
    📋 Copy-Ready Instructions
    When user asks about Express or Standard shipping, apply the following consolidation: - "Same Day", "First Class" are mapped to "Express" Shipping - "Standard Class", "Second Class" are mapped to "Standard" Shipping When user uses terms such as "express", "standard", "express vs standard", "shipping comparison", apply this mapping before generating any chart or insight. Always use the consolidated group name in chart labels and insight text, not the raw column values.
    💬 Examples
    Example 1
    USER
    "Compare profit for Express vs Standard shipping"
    AGENT
    Agent maps Same Day + First Class → Express, Standard Class + Second Class → Standard. Shows Avg(Profit) per group as a Bar Chart.
    Example 2
    USER
    "What percentage of orders use Express shipping?"
    AGENT
    Agent counts orders where Ship Mode IN (Same Day, First Class) as Express, everything else as Standard. Shows Pie Chart.
    Example 3
    USER
    "Show average delivery time by shipping type"
    AGENT
    Agent applies the same grouping, computes Avg(Delivery_Days) per group. Bar Chart with insight.
    💡 Practical Tips
    1.Map EVERY raw value — leave no gaps
    2.Each raw value goes to exactly one group
    3.List trigger words/synonyms
    4.Show group name in charts, not raw values
